Comment ça marche

Pour pouvoir passer commande sur notre site vous avez deux possibilité, dépandant du choix que vous souhaiter.
Pour un (ou plusieurs) menu(s) simple vous trouverez dans la section Menu, une liste comportant les plats que nous
proposons ainsi que leur tarifs vous aurez juste a constituer votre panier. Une fois votre commande terminer vous
recevrez un mail comprenant: la confirmation de votre commande, la facture liée à votre commande et les details concernant
votre livraison. Pour toute demande concernant un événement nous vous proposons, un formulaire de contact pour toute
demande de devis avec tous les élément necessaire pour l'effectuer.
